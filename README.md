# 🍔 Food Delivery Web Application

A full-stack **Food Delivery Web Application** built using the **MERN Stack**. Users can browse food items, add products to their cart, place orders, make payments using Stripe, and track their orders. An Admin Panel is also included for managing food items and customer orders.

## ✨ Features

### 👤 User Features
- User Registration and Login
- JWT Authentication
- Browse Food Items
- Add and Remove Items from Cart
- Persistent Cart Data
- Place Food Orders
- Secure Online Payment using Stripe
- View Previous Orders
- Track Order Status

### 🛠️ Admin Features
- Add New Food Items
- View Food List
- Remove Food Items
- View All Customer Orders
- Update Order Status
  - Food Processing
  - Out for Delivery
  - Delivered
